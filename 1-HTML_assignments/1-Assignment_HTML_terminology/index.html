HTML - Terminology & Definitions

  1. What do HTML and CSS stand for?
    • HTML stands for Hyper Text Language.
    • HTML is the most basic building block of the web. It defines the meaning and structure of web content.

  2. Would you rather use HTML or CSS, for putting paragraphs of text on a webpage?
    • I rather use HTML.

  3. Would you rather use HTML or CSS, to change the font and background color of a button?
    • I rather use CSS.

  4. What is an HTML tag?
    • HTML tags are the keywords on a webpage that define how your web browser must format and display your webpage.
    • Almost all tags contain two parts, an opening, and a closing tag.

  5. What are the three parts of an HTML element?
    • The opening tag (or star tag) marks where the element's content begins.
    • The closing tag (or end tag) marks the end of the element's content. The closing tag is identical to the opening tag with the addition of a forward slash after the first angle bracket.
    • The content is placed between the element's opening tag and closing tags.

  6. What is the purpose of the doctype declaration?
    • The doctype declaration is an instruction to the web browser about what version of HTML the page is written is.

  7. What is the HTML element?
    • An HTML element is defined by a start tag, some content, and an end tag.
    • HTML elements tell the browser how to display the content.

  8. What is the purpose of the head element?
    • The head element conatins meta information about the HTML page.

  9. What is the purpose of the body element?
    • The body element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

  10. How do you create a paragraph in HTML?
    • To create a paragraph, first you type the paragraph star tag: < p >.
    • Then type the paragraph's text.
    • At the end of the paragraph's text, type the end tag: < /p >.

  11. i rather use HTML. How do you create a heading in HTML?
    • Heading are created with the < h1 > to < h6 > tags. < h1 > defines the most important heading. < h6 > defines de least important heading.

  12. How many different levels of headings are there and what is the difference between them?
    • There are six levels of headings. The level one is teh most importante and the level six is the least important.

  13. What element should you use to make text bold and important?
    • I should use the < strong > element.

  14. What element should you use to make text italicized to add emphasis to it?
      I should use the < em > element.

  15. What relationship does an element have with any nested elements within it?
    • When an HTML element is nested inside another element, it is considered a child of the nesting element.

  16. What relationship do two elements have if they are at the same level of nesting?
    • The two elements are siblings.

  17. How do you create HTML comments?
    • You can add comments in HTML using the syntax: < !-- comment -->.

  18. What HTML tag is used to create an unordered list?
      The < ul > tag.

  19. What HTML tag is used to create an ordered list?
    • The < ol > tag.

  20. What HTML tag is used to create list items within both unordered and ordered lists?
      The < li > tag.

  21. What element is used to create a link?
    • The < a > tag.

  22. What is an attribute?
    • Attributes provide additional information about elements.
    • Atributes are always specified in the start tag.

  23. What attribute tells links where to go?
    • The attribute "src=".

  24. What is the difference between an absolute and relative link?
    • An absolute link provides the complete location infomation of your site, while a relative link contains the location following your domain.

  25. Which element is used to display an image?
    • The elemente < img >.

  26. What two attributes do images always need to have?
    • The "src" and "alt" attributes.

  27. How do you access a parent directory in a file path?
    • Using .. and /.

  28. What are the four main image formats that you can use for images on the web?
    • JPEG, PNG, SVG and GIF.
